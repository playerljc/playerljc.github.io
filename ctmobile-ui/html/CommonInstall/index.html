安装

    
      npm install @ctmobile/ui --save-dev

使用

引入js


    import {Accordion} from '@ctmobile/ui';

按需引入 js

  • 使用 babel-plugin-import

    {
      "plugins": [
        ["import", { libraryName: "@ctmobile/ui"}]
      ]
    }

引入less

@import "../node_modules/@ctmobile/ui/lib/ctmobile-ui.less"
